# UAV Mission Environment

A Python package for UAV mission environments.

## Installation

### From Source

To install the package in development mode:

```bash
pip install -e .
```

To install the package:

```bash
pip install .
```

### From PyPI (when published)

```bash
pip install uav_mission_env
```

## Usage

```python
import uav_mission_env

print(uav_mission_env.__version__)
```

## Development

To install in editable mode with development dependencies:

```bash
pip install -e .
```
